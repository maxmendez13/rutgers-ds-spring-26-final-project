# rutgers-ds-spring-26-final-project
For Professor Hongyi Wang's class


To properly run the code:
- Make sure that miniconda is installed in the coding environment
- python -m pip install -U pip
- install pandas: pip install pandas scikit-learn
- install numpy: pip install numpy
- install seaborn: pip install seaborn
- install matplotlib.pyplot: python -m pip install -U matplotlib

Download the Kaggle dataset that is used to train the data:
https://www.kaggle.com/datasets/emineyetm/fake-news-detection-datasets?resource=download

Unzip the files and place Fake.csv and True.csv into the file folder
